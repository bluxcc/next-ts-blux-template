# Next v16 + TypeScript + Blux Template

A minimal Next starter with Blux already integrated.
To get started a developer must clone the repository,
install dependencies, set the Blux app id in the environment
(from https://dashboard.blux.cc), and start the development server.

## Quick start

- Clone the repository:

- Install dependencies:

  ```bash
  npm i
  ```

- Add the Blux app id:
  - Obtain the app id from https://dashboard.blux.cc.
  - Copy `.env.example` to `.env` and set the `NEXT_PUBLIC_BLUX_APP_ID` value.

- Start the development server:

  ```bash
  npm run dev
  ```
